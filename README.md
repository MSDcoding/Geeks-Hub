# 🎮 Geeks Hub: The Escape Room (Working Title)

> 🚧 **Status:** Work In Progress (WIP) - Currently under active development.

Welcome to the repository of **Geeks Hub**, a first-person horrorPsycho/escape room experience built in Unity. 

## 📖 The Story
You find yourself trapped inside "Geeks Hub," a tech institute that has suddenly turned into a technological nightmare. The doors are locking on their own, the lights are glitching, and a mysterious AI voice is broadcasting from the screens. You are not alone—something is watching you from the dark corridors. Armed with only your flashlight and your wits, you must interact with the environment, solve puzzles, and find an escape route before you become a permanent part of the Hub.

## 🛠️ My Role as a Game Developer
This is a team project where I am responsible for the core gameplay mechanics, logic, and UI. Here is what I have developed so far:
* **Advanced Interaction System:** Developed a robust Raycast-based system allowing the player to interact with objects (e.g., picking up items, opening doors) using the `[E]` key.
* **Dynamic Trigger Events:** Programmed invisible trigger zones that activate cinematic sequences (doors slamming shut, lights turning on, AI voice lines playing) to build horror tension.
* **User Interface (UI):** Designed and scripted a fully functional Main Menu and Pause Menu with scene transitions, time-scaling, and resolution handling.
* **Physics & Colliders:** Configured Unity's physics engine (Rigidbodies, Box/Capsule Colliders) to ensure realistic player movement and object boundaries.
* **Scene Management:** Handled smooth screen fading and loading between the Main Menu, the escape rooms, and different game levels.
